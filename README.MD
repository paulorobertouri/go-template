# go-template

Go API template using Echo with JWT authentication, Swagger docs, and automated tests.

## Stack

- Go 1.22+
- Echo
- swaggo/echo-swagger
- golang-jwt/jwt
- Testify

## Endpoints

- `GET /health`
- `GET /docs`
- `GET /v1/public`
- `GET /v1/customer`
- `POST /v1/auth/login`
- `GET /v1/private`

## JWT Contract

Login returns:

- Body: `{"token":"..."}`
- Header: `Authorization: Bearer <token>`
- Header: `X-JWT-Token: <token>`

## Commands

- `make install` ‚Äî download dependencies
- `make install-dev` ‚Äî install dev tools
- `make run` ‚Äî run the application
- `make test` ‚Äî run tests
- `make test-coverage` ‚Äî run tests with coverage report
- `make docker-build` ‚Äî build Docker image
- `make docker-test` ‚Äî run tests in Docker
- `make docker-curl-test` ‚Äî run curl integration tests in Docker

## Environment Variables

- `PORT=8000`
- `JWT_SECRET=your-super-secret-jwt-key-at-least-32-characters-long-for-hs256`
- `JWT_ALGORITHM=HS256`
- `JWT_EXPIRATION=3600`
